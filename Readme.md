# SimTools Showcase: Stochastische Simulation
### Studentische Projekte im Studiengang Social Data Science & Communication
**Technische Hochschule Nürnberg Georg Simon Ohm** (https://www.th-nuernberg.de/)

---

## Über diesen Kurs

Dieses Repository dient als Sammelstelle für die Projektergebnisse aus dem Modul **"SimTools"**. 

In diesem Semester haben wir den Fokus von der reinen Theorie auf die praktische Anwendung gelegt. Die Studierenden hatten die Aufgabe, erworbene Fähigkeiten im Bereich der stochastischen Simulation (Monte-Carlo-Methoden, Warteschlangentheorie, Agentenbasierte Modellierung) auf eigene, frei gewählte Fragestellungen anzuwenden.

Die Ergebnisse umfassen:
* Reproduzierbaren Code (Python)
* Technische Berichte
* Visualisierungen komplexer Zusammenhänge

Dieses repository ist eine Linksammlung für den abgegebenen Code
